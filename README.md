# minerva-roulette

Tells you who is in charge of MIN tickets this week.

A Nuxt 4 single-page application featuring a carnival-style spinning wheel.
Every Thursday at 09:30 CEST the name rotates through `[Kris, Gilles, Tom, Alex]`,
starting from **Kris** on **2026-06-11**.

## Setup

Make sure to install dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

Check out the [deployment documentation](https://nuxt.com/docs/getting-started/deployment) for more information.
